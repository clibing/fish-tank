### 一、鱼缸

半自动的小型DIY鱼缸

### 二、功能实现

#### 功能介绍

* [ ] 吸粪便等杂物
* [ ] 夜间照明
* [ ] 供氧
* [ ] 太阳能供电
* [ ] 绿植
* [ ] 晒台
* [ ] 其他

#### 软件部分

* [ ] 自动配网
* [ ] 固件升级
* [ ] mqtt-ssl通道
* [ ] 指令下发
* [ ] 温度、湿度上报
* [ ] 今日天气

#### 电路周围

* [ ] OLED 显示如下
  + 温度、湿度
  + 水温
  + 今日天气
  + 当前运行状态(Running,Error)
* [ ] 温湿度模块
  采集当前空气下的温度、湿度
* [ ] 温敏电阻采集水温  
* [ ] 摄像头采集照片(Esp8266不支持,内存太小，cpu频率低等原因)
